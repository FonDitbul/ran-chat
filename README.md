
# ran-chat

## Auth

회원탈퇴

profile, 프로필 변경, 이름변경, 프로필 이미지 변경, 소개 메시지 

OAuth2.0 
JWT 토큰

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
